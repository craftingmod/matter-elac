## Matter-Elac

![Project Thumbnail](./public/thumb.avif)

Matter bridge for ELAC DS-A101-G.

Currently, only DS-A101-G is supported.

## SmartThings Edge Driver

https://bestow-regional.api.smartthings.com/invite/d4294nK49bMo
